# Lookie Releases

This repository hosts the public download binaries for **Lookie**. It does not contain source code — only the release artifacts (macOS `.dmg` and Windows `.exe`).

## What is Lookie?

That meeting could've been a Lookie.

Lookie is a desktop app for remote teams. Record your screen, share a link, and let people watch and respond on their own time — no meetings, no long emails. Just show what you mean.

- **Record** your screen from your desktop
- **Share** a link with your team
- **Respond** on your own time with comments and reactions
- Videos **expire automatically** so your drives stay clean

Learn more at [lookie.space](https://lookie.space).

## Downloads

Get the latest version from the [Releases page](https://github.com/pixelspace-studio/lookie-releases/releases):

- **macOS** (Apple Silicon): `Lookie-<version>-arm64.dmg`
- **Windows** (x64): `Lookie-Setup-<version>.exe`

Only the two most recent versions are kept per platform.
